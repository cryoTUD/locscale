# Running LocScale via CCPEM Doppio

```LocScale 2``` can be run via the [CCPEM Doppio](https://www.ccpem.ac.uk/docs/software/) interface. CCPEM Doppio support all four ```LocScale 2 modes``

### LocScale in CCPEM tutorial

In this example we will use the Rattus norvegicus TRPV1 channel [EMDB 5778](https://www.ebi.ac.uk/emdb/EMD-5778) from the [EMDB map
and model challenge](https://zenodo.org/records/1185426).
